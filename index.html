COMUNICAÇÃO SPI ENTRE MICROCONTROLADORES ARM CORTEX-M4 TIVA C SERIES TM4C123GH6PM

Coautor - II CONAPESC - 2017

Esse trabalho deixa clara a importância das pessoas que buscam o conhecimento e entendimento da tecnologia nessa época em que ela se encontra como a área mais promissora da logística mundial, entre várias empresas abertas ao mercado. O uso de microcontroladores, mais completos e complexos que microprocessadores, mostram como o avanço tecnológico pode estar cada vez mais presente no nosso cotidiano como em aparelhos eletrodomésticos, sistemas de segurança e monitoramento, de maneira eficiente e podendo ser facilmente adaptada através do estudo dedicado ao microcontrolador usado sem encapsulamento, para que o uso de bibliotecas do compilador seja feita de forma consciente e alterações sejam mais viáveis. O artigo mostra a função, a importância e o uso da comunicação Serial Peripheral Interface entre as placas da Tiva C Series, TM4C123GH6PM, deixando claro todos os procedimentos necessários para sua conclusão no intuito de criar um aprendizado eficaz e proativo do assunto.

INVERSOR DE ONDA QUADRADA PARA SISTEMAS FOTOVOLTAICOS DE PEQUENO PORTE

Coautor - II CONAPESC - 2017

Este trabalho foi desenvolvido com objetivo construir um equipamento denominado inversor para um sistema fotovoltaico de pequeno porte, voltado para uso em iluminação de pequenas residências. A cada dia que passa o consumo de energia aumenta, assim como a busca por outras fontes dela. Uma fonte abundante, barata e limpa é a energia solar. Apesar do Brasil possuir um dos maiores potenciais para geração desse tipo de energia, ela corresponde a menos de 1% da matriz energética nacional. Um dos maiores motivos que impedem o crescimento do uso de energia solar é o elevado custo dos painéis solares e outros equipamentos. Um desses equipamentos é o inversor cuja função é converter a tensão contínua dos painéis solares para alternada que é a utilizada pelos aparelhos de uma residência. Com base nisso, é proposto um inversor de onda quadrada de menor custo para incentivar o uso de energia solar. Este tipo de inversor pode ser utilizado para alimentar lâmpadas fluorescentes compactas, eliminando a necessidade de energia elétrica proveniente da rede para iluminação residencial.

MÁQUINA CNC PARA CONFECÇÃO DE PLACAS DE CIRCUITO IMPRESSO UTILIZANDO AS PLATAFORMAS DE CÓDIGO ABERTO ARDUINO E GRBL

Coautor - II CONAPESC - 2017

Este artigo apresenta o desenvolvimento de uma máquina com controle numérico computadorizado, CNC, utilizando plataformas de código aberto e voltada para a confecção de placas de circuito impresso, PCI, para projetos de pesquisa. Foram utilizadas na concepção desse projeto as plataformas de código aberto Arduino e GRBL o que possibilitou a construção desse tipo de máquina com um menor custo. Na metodologia é apresentada como foi realizada a construção da estrutura mecânica e do circuito eletrônico de controle. Além disso também é explicado como a máquina é controlada por um computador utilizando softwares desenvolvidos pela comunidade GRBL.
